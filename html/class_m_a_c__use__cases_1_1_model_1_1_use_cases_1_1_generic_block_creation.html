inlinestatic

Creates a Function Block (FB) in the target PLC's program blocks folder.

This method performs the following steps:

  1. Creates a block call with default input values (Input1=1, Input2=0)
  2. Sets up a network with titles and the block call
  3. Creates a standard FB
  4. Configures the FB with network, attributes, comments, and titles
  5. Generates the final block in either XML or SCL format
Parameters
blockNameName of the FB to create
instanceDbNameName of the instance data block
programmingLanguageProgramming language for the FB (supports LAD, FBD, SCL)
plcDeviceTarget PLC device where the FB will be created
Exceptions
ArgumentNullExceptionThrown when blockName, instanceDbName, or plcDevice is null
ArgumentOutOfRangeExceptionThrown when programming language is not supported

Creating a standard LAD function block:

CreateFB("MyFB", "MyFB_DB", ProgrammingLanguage.LAD, plcDevice);
