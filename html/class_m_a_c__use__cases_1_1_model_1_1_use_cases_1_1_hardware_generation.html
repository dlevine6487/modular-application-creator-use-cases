string name )
inlinestatic

Creates or retrieves an HMI device in the TIA Portal project.

Parameters
projectThe TIA Portal project object where the HMI device should be created or found.
nameThe name identifier for the HMI device.
Returns
The HmiTarget object representing the software container of the HMI device.

This method performs the following steps:

  • Searches for an existing HMI device with the specified name
  • If not found, creates a new HMI device with default configuration (6AV2 125-2JB23-0AX0/17.0.0.0)
  • Retrieves the software container for the HMI Runtime
  • Returns the HMI target software object
Exceptions
ExceptionThrown when the HMI software container cannot be accessed or created.

Creates or retrieves a device in the TIA Portal project based on the specified parameters.

Parameters
projectThe TIA Portal project object where the device should be created or found.
typeIdentifierThe device type identifier in the format "OrderNumber:XXXX/Version[/Type]".
nameThe unique identifier name for the device within the project.
deviceNameThe display name of the device in the TIA Portal.
Returns
The Device object representing the created or retrieved device.

The typeIdentifier parameter should follow the format:

  • For standard devices: "OrderNumber:1234567-1AX00/V1.0"
  • For typed devices: "OrderNumber:1234567-1AX00/V1.0/Type1" If a device with the specified name already exists, it will be returned. Otherwise, a new device will be created with the specified parameters.
Exceptions
ExceptionThrown when the device cannot be created with the specified parameters.
