Creates a new tag table or retrieves an existing one in the specified PLC device.

Parameters
plcDeviceThe PLC device where the tag table should be created or retrieved
tableNameThe name for the tag table
Returns
A ControllerTags object representing the new or existing tag table

If a tag table with the specified name already exists, it will be returned instead of creating a new one. The returned tag table can be used to add, modify, or remove tags.

var tagTable = CreateTagTable(myPlcDevice, "ProcessTags");

Creates a new tag table or retrieves an existing one in the specified software unit.

Parameters
softwareUnitThe software unit where the tag table should be created or retrieved
tableNameThe name for the tag table
Returns
A ControllerTags object representing the new or existing tag table

This overload creates/retrieves a tag table within a specific software unit rather than at the PLC device level. If a tag table with the specified name already exists in the software unit, it will be returned instead of creating a new one.

var tagTable = CreateTagTable(mySoftwareUnit, "ModuleSpecificTags");
